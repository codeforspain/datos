Code for Spain es una iniciativa sin ánimo de lucro para mejorar nuestras ciudades y comunidades aprovechando el open data y el potencial de la tecnología. Te invitamos a participar, todos los perfiles son necesarios.

Repositorio de datos abiertos

El primer proyecto de Code for Spain consiste en tener 10 datasets muy usados por desarrolladores y periodista en formatos estándar.
